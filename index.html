Wilma

Wilma is HTTP/HTTPS proxy and stub solution, written in Java. It consists of two applications:

  • Wilma application
  • Wilma Message Search application

Wilma application acts as a proxy between two components those are communicating over HTTP/HTTPS. It logs the messages, and makes it possible to stub responses based on configuration defined in XML file(s). Also configurable via plugins.

Wilma Message Search application provides high performance searching of the request-response pairs those were logged by Wilma application.
